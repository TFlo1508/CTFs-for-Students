Session Handling - Datei 2

Session Handling kann in Browsern mithilfe von JavaScript und dem localStorage Mechanismus erreicht werden.
Wenn Sie eine Session-Variable hinzufügen/erstellen ist diese existent, so lange wie Sie die Browserinstanz offen haben, also bis alle Browser-Fenster geschlossen werden.
Dies ist insbesondere hilfreich, wenn Sie Informationen über längere Zeit "behalten" möchten und nicht mit URL Parametern arbeiten wollen oder können.

Weitere Infos:
https://developer.mozilla.org/de/docs/Web/API/Window/localStorage
